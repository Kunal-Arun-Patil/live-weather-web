🌦️ LiveWeather - Real-Time Weather Application
📌 Overview

LiveWeather is a modern weather web application that provides real-time weather updates, air quality information, wind direction tracking, and weather forecasts for cities worldwide.

🚀 Features
🔍 City Search with Autocomplete
📍 Automatic Geolocation Detection
🌡️ Current Weather Information
💨 Wind Direction Compass
🌿 Air Quality Index (AQI)
⏰ 24-Hour Forecast
📅 5-Day Forecast
🎨 Dynamic Weather Themes
🌍 Global City Support
🔄 Celsius / Fahrenheit Toggle
✨ Animated Background Effects
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Node.js
OpenWeather API
